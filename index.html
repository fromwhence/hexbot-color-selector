About and how to use:

  • Colors are generated at random using the Hexbot API.
  • Select the refresh icon to view a new set of colors. The larger the viewing area, the more colors will be displayed.
  • To save a hex code, select the color's plus sign.
  • The hex codes of saved colors can be copied to the clipboard by using the clipboard button.
  • Colors can be removed individually from favorites by selecting the color tile or removed all at once by using the trashcan.
